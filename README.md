## ataxx.el

ataxx.el is an implementation of the
[ataxx](https://en.wikipedia.org/wiki/Ataxx) game in Emacs Lisp.

### How to play

On Emacs, run:

    M-x load-file RET caminho/para/o/ataxx.el
    M-x ataxx

For more information about key bindings, run `M-x describe-mode`.
